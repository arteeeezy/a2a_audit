# Simplify - Code Quality Review

Usage: /simplify [focus]

Process:
1. Spawns three review agents in parallel
2. Aggregates their findings
3. Applies fixes automatically

Optional focus parameter for specific concerns
Example: /simplify focus on memory efficiency

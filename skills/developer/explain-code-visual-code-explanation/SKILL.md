# Explain Code - Visual Code Explanation

Usage: /explain-code or automatically triggered when asking how code works

Structure:
1. Start with an analogy - Compare code to everyday life concepts
2. Draw a diagram - Use ASCII art to show flow, structure, or relationships
3. Walk through the code - Step-by-step explanation
4. Highlight a gotcha - Common mistakes or misconceptions

Style: Conversational explanations, multiple analogies for complex concepts

Example setup:
- Create directory: ~/.claude/skills/explain-code
- Create SKILL.md with frontmatter and instructions
- Test by asking 'How does this code work?' or using /explain-code src/auth/login.ts

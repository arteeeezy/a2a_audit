# Debug - Session Troubleshooting

Usage: /debug [description]

Features:
- Enables debug logging for current session
- Debug logging is off by default unless started with claude --debug
- Captures logs from invocation point forward
- Optional description parameter to focus analysis on specific issues


## Steps

1. Enables debug logging for current session
2. Debug logging is off by default unless started with claude --debug
3. Captures logs from invocation point forward
4. Optional description parameter to focus analysis on specific issues
